##Steps To Use GitHub Repositories
Ensure Git is installed. 

##Git Repository
Create the git repository, for example:
https://github.com/muhammadshumail/Mathematica.git

##Local Repository
1.	Launch command prompt
Microsoft Windows [Version 10.0.26200.7462]
(c) Microsoft Corporation. All rights reserved.

2. Change to the path where you would like to create the corresponding local repository
C:\Users\muham>cd C:\Users\muham\Documents\Quantum Information Technology

3. Clone the remote repository to the local location
C:\Users\muham\Documents\Quantum Information Technology>git clone https://github.com/muhammadshumail/Mathematica
Cloning into 'Mathematica'...

Receiving objects: 100% (7/7), done.

4. Your local repository has been created with the same name. Change path to it.
C:\Users\muham\Documents\Quantum Information Technology>cd Mathematica

5. Just to check the short name for the remote repository.
C:\Users\muham\Documents\Quantum Information Technology\Mathematica>git remote -v
origin  https://github.com/muhammadshumail/Mathematica (fetch)
origin  https://github.com/muhammadshumail/Mathematica (push)

6. Optional. I did it the first time. But you only have to do after you make changes.
C:\Users\muham\Documents\Quantum Information Technology\Mathematica>git push -u origin main

7. Add your files to the local repository. Edit them. Update them. Things like that. Then add them.
